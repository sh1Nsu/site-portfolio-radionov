    /3D МОДЕЛЬ ПЕРСОНАЖА ДЛЯ МОБИЛЬНОЙ ИГРЫ

    проект создавался для интерфейсов мобильной игры в жанре гасяпон. Персонаж создавался по концепту Шрека. Работу можно использовать для реальной мобильной игры. Каркас персонажа создавался при помощи скульптинга в программе Blender. Также была создана локация, представляющая собой "ареал обитания" персонажа.
